# Content Guide

## Audience
- Companies needing senior software development help
- Teams needing quality-focused implementation and maintainable systems

## Core Message
I help companies build high-quality, maintainable software.

## Tone
- Clear
- Professional
- Practical
- No hype

## Recommended Home Page Sections
1. Intro:
- Name
- Role: Independent software consultant
- One-sentence value proposition

2. Services (short):
- Example: Full-stack development
- Example: Architecture and technical leadership
- Example: Legacy modernization and maintainability improvements

3. Selected Experience:
- Company/client name (if shareable)
- Role or contribution
- Time period
- 1-2 bullets on measurable impact

4. Contact:
- Email
- LinkedIn (optional)
- Location/time zone (optional)

## Keep It Current
- Review CV items at least quarterly
- Remove old or low-relevance entries
- Prefer fewer, stronger examples over long lists
